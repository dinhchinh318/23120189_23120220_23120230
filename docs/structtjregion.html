int tjregion::h

The height of the cropping region. Setting this to 0 is the equivalent of setting it to the height of the source JPEG image - y.

◆ w

int tjregion::w

The width of the cropping region. Setting this to 0 is the equivalent of setting it to the width of the source JPEG image - x.

◆ x

int tjregion::x

The left boundary of the cropping region. For lossless transformation, this must be evenly divisible by the iMCU width (see #tjMCUWidth) of the destination image. For decompression, this must be evenly divisible by the scaled iMCU width of the source image.

◆ y

int tjregion::y

The upper boundary of the cropping region. For lossless transformation, this must be evenly divisible by the iMCU height (see #tjMCUHeight) of the destination image.
